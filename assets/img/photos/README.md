# 10인 공식 이미지 (L1 몽타주 · 도감 뒷면 · ⑤ 결과)

- `*.jpg` = **사용본**. 400×480(5:6) 흑백, 표시 200×240의 2배. 얼굴 기준 수동 크롭.
- `src/` = 원본(내려받은 그대로, 색·원크기 유지). 재크롭 필요하면 여기서.
- `alt/` = 한용운 **실사진** 후보(아래 ⚠ 참조).

출처·저작권 = `docs/image_sourcing.md` (전원 PD). 크레딧 표기 예시도 거기.

## ⚠ 미해결 — 2장은 사진이 아니라 영정(그림)
`image_sourcing.md` 첫 줄 "10인 전원 PD **사진** 확보"는 이 둘에 대해 **사실이 아님**(2026-07-15 확인).

| 인물 | 시트 기록 | 실제 |
|---|---|---|
| 한용운 | "한복 정면 초상 · PD-Japan-**oldphoto**" | **영정(채색화).** 출처=홍성군청 `historyHuman_img01.gif` |
| 윤희순 | "사진인지 영정인지 **불확실**"(보류) | **영정 확정.** 연필·담채 전신화, 얼굴 아주 작음 |

**왜 문제인가**: 엔딩 B안 = "본편=목판·익명 / 엔딩=**실사**·실명, 스타일이 걷히는 것이 반전". 그림이면 그 두 명에게 반전 문법이 안 섬. 시트는 이미 윤봉길에 **"표준영정(그림) 금지"** 규칙을 세워둠 — 같은 잣대면 이 둘도 걸림. 한용운은 캡스톤.

**대안 현황**
- 한용운: 실사진 있음, 전부 작음 → `alt/` 에 보관
  - `alt_10_hanyongun_photo_181x239.jpg` (만해 초상, 젊은 시절)
  - `alt_10_hanyongun_photo1937_134x209.jpg` (1937년)
  - ※커먼즈 "Han Yong-un (cropped) 499×587"은 **동상 사진**이라 무용
- 윤희순: 커먼즈 전체에 이 영정 1장뿐. 시트의 대안=친필 가사집
- **보훈부 공훈전자사료관은 확인 못 함**(검색이 JS 구동이라 스크래핑 불가). "없음"이 아니라 "미확인" — 사람이 브라우저로 봐야 함
